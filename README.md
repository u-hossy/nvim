# nvim

setting of Neovim

## How to use

Put this repo in `~/.config` as `nvim`

```bash
mkdir ~/.config
cd ~/.config
git clone <git-repo>
```

## Formatters

### Lua

- [stylua](https://github.com/JohnnyMorganz/StyLua)

### Yaml

- [yamlfmt](https://github.com/google/yamlfmt)
    - Installs at `$HOME/go/bin/`
